Cafe API REST

Esta es un API REST DEL CAFE

Potenciando Cafes

Ver todos los cafés

Método: GET

Endpoint: /api/cafes

Obtener café por ID

Método: GET

Endpoint: /api/cafes/:id

Buscar café por país

Método: GET

Endpoint: /api/cafes/pais/Italia

Filtrar cafés por peso

Método: GET

Endpoint: api/cafes/peso/250g

Filtrar cafés por categoría

Método: GET

Endpoint: api/cafes/categoria/Espresso

Obtener Café por nombre

Método:

GET

Endpoint /api/cafes/nombre/Latte

Crear un café

Método: POST

Endpoint: /api/cafes

Actualizar un café

Método: PUT

Endpoint: /api/cafes/:id

Eliminar un café

Método: DELETE

Endpoint: /api/cafes/:id
